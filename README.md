# NBA-Game-Analysis
Data science project using NBA game data to uncover player impact, analyze game momentum, and build predictive models with real-world datasets.
## 📂 Dataset

The dataset used in this project is hosted on Google Drive due to GitHub file size limits.

👉 [Download Dataset](https://drive.google.com/file/d/1VlmRGgAuRq_1F_62xoct5cD9WN36XyBS/view?usp=drive_link)

### Details
- Source: NBA play-by-play data  
- Format: CSV  
- Size: ~21 MB  
- Description: Contains game events, timestamps, and player statistics used for analysis and modeling.
